# Projeto MultiMed Corrigido

Projeto didático com HTML, CSS, JavaScript no front-end e API Node.js/Express no back-end.

## Como executar

```bash
npm install
npm start
```

Depois acesse:

```text
http://localhost:3000
```

## Sobre o JavaScript

As funções principais de cada página foram colocadas dentro do próprio HTML, no final do arquivo, seguindo o formato didático:

```html
<script>
    function exemplo() {
        // código da página
    }
</script>
```

O arquivo `public/js/api.js` permanece separado porque é usado por várias páginas para fazer as chamadas `fetch` para a API.
